# AWS Settings URL

## ChipID 
http://192.168.4.1/chipID

## RootCA String
http://192.168.4.1/rootCA?ca=ROOTCA_STRING

## Certificate String
http://192.168.4.1/cert?cert=CERTIFICATE_STRING

## Private String
http://192.168.4.1/privateKey?key=PRIVATEKEY_STRING

## Endpoint
http://192.168.4.1/endpoint?endpoint=ENDPOINT_STRING
